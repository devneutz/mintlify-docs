ShowUpFor's core product — events, shifts, signups, teams, statistics,
reminders, manage links, and privacy/retention handling — is free for every
club, with no time limit. Pro is a small set of paid upgrades on top of
that, not a cut-down free tier.

<Note>
  The **Billing** page only appears in the admin nav once your club's
  ShowUpFor deployment has billing turned on. If you don't see it, your club
  is simply on the default (Free) plan and upgrading isn't available yet —
  ask whoever runs your deployment.
</Note>

## Free vs. Pro

| | Free | Pro |
| --- | --- | --- |
| Events, shifts, signups, statistics, teams | ✓ | ✓ |
| Logo branding | ✓ | ✓ |
| Confirmation, cancellation & reminder emails | ✓ | ✓ |
| Manage links, privacy notice, data retention | ✓ | ✓ |
| Admin seats | 2 | Unlimited |
| Custom accent color | Default color only | Custom |

## Upgrading

On the **Billing** page, your club's plan and available upgrades are shown
via an embedded pricing table. Changes apply to the whole club immediately
and take effect for every admin.
