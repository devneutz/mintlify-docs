Der Kern von ShowUpFor — Anlässe, Schichten, Anmeldungen, Teams, Statistik,
Erinnerungen, Verwaltungslinks sowie Datenschutz und Aufbewahrung — ist für
jeden Verein zeitlich unbegrenzt gratis. Pro ist eine kleine Auswahl an
kostenpflichtigen Erweiterungen obendrauf, keine abgespeckte Gratis-Version.

<Note>
  Die Seite **Abo** erscheint in der Admin-Navigation nur, wenn für euer
  ShowUpFor-Deployment das Abo-Modul aktiviert ist. Fehlt sie, ist euer
  Verein einfach auf dem Standard-Plan (Free) und ein Upgrade ist noch nicht
  verfügbar — frag die Person, die euer Deployment betreibt.
</Note>

## Free vs. Pro

| | Free | Pro |
| --- | --- | --- |
| Anlässe, Schichten, Anmeldungen, Statistik, Teams | ✓ | ✓ |
| Logo-Branding | ✓ | ✓ |
| Bestätigungs-, Stornierungs- und Erinnerungs-E-Mails | ✓ | ✓ |
| Verwaltungslinks, Datenschutzhinweis, Aufbewahrungsfrist | ✓ | ✓ |
| Admin-Plätze | 2 | Unbegrenzt |
| Eigene Vereinsfarbe | Nur Standardfarbe | Frei wählbar |

## Upgraden

Auf der Seite **Abo** werden der Plan eures Vereins und verfügbare Upgrades
über eine eingebettete Preistabelle angezeigt. Änderungen gelten sofort für
den ganzen Verein und für alle Admins.
